    Начиная со второй строки — данные оборудования в следующем порядке:
    A B C D
    Инв. номер Наименование * Дата ввода в эксплуатацию МОЛ
    ИНВ-001 Станок токарный 01.03.2022 Иванов И.И.
    * — обязательное поле. Дата в формате дд.мм.гггг. Если инв. номер уже есть в базе — строка будет пропущена.
